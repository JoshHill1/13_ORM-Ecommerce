# 13_ORM-Ecommerce

## Description

This assignments challenge was to build the back end for an e-commerce site. I worked with Express.js API and configure it to use Sequelize to interact with a MySQL database.

## Usage

mysql -u root -p
source db/schema.sql
source db/seeds.sql
quit
npm start
